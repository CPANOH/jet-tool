# JET · 분개 검증 툴 (Journal Entry Testing)

> **▶ 바로 실행 (설치 불필요, 클릭만 하면 됩니다)**
> - 🟢 **GitHub Pages:** https://cpanoh.github.io/jet-tool/
> - 🔷 **Vercel:** https://jet-tool.vercel.app

회계감사기준서 **240(경영진의 내부통제 무력화 위험)**에 대응하는 분개 검증(JET) 도구입니다.
빅4가 수행하는 Journal Entry Testing의 흐름 — **데이터 입수 → 위험 특성 필터링 → 이상 전표 식별 → 감사조서화** — 을 브라우저에서 체험할 수 있습니다.

## 주요 기능

- **산업별 가상 분개장 생성** — 일반 제조·상거래 / 조선 / 건설 / 반도체
  - 각 산업의 주요 거래와, 그 산업 특유의 부정 시나리오(예: 조선·건설의 진행률 조작, 반도체의 R&D 자본화·재고평가손실 환입)를 포함
- **Red flag 위험 스코어링** — 업무외 시간·주말·공휴일 전기, 소급 입력, 결산 임박, 라운드 금액, 승인한도 직하, 적요 공란, 자기승인, 비경상 계정 조합 등 13개 규칙으로 이상 전표를 점수화
- **감사인 관점 해석** — 각 이상 전표에 위험 사유와 후속 절차 제안 표시
- **벤포드 법칙 검정** — 금액 첫 자리 분포로 데이터 전반의 조작 가능성 진단
- **감사조서(CSV) 내보내기** — 수행절차·입수증빙·결론을 입력해 엑셀 조서로 출력

## 사용 방법

1. **① 데이터** — 산업과 주요 거래를 고르고 "가상 분개장 생성"
2. **② 위험 분석** — 승인 한도·결산일 설정 후 "위험 분석 실행" → 이상 전표를 위험점수 순으로 확인, 전표별 감사 코멘트 입력 → 감사조서 내보내기
3. **③ 대시보드** — 규칙별 적발 건수, 벤포드 분포
4. **④ 규칙 설명** — 적용된 red flag와 감사상 의미

## 실행 환경

단일 HTML 파일입니다. 위 링크로 바로 실행하거나, 저장소를 내려받아 `index.html`을 브라우저에서 열면 됩니다. (별도 설치·빌드 불필요, 오프라인에서도 동작)
